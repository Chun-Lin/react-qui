


### `light` Theme

```js
<div>
    <Button theme="light" shape="square">Button</Button>
    <Button theme="light" shape="rounded">Rounded</Button>
    <Button theme="light" shape="square" disabled>Square</Button>
    <Button theme="light" shape="rounded" disabled>Square</Button>
</div>
```

### `dark` Theme

```js
<div>
  <div style={{ backgroundColor: 'black', padding: '10px' }}>
    <Button theme="dark" shape="square">Button</Button>
    <Button theme="dark" shape="rounded">Rounded</Button>
    <Button theme="dark" shape="square" disabled>Square</Button>
    <Button theme="dark" shape="rounded" disabled>Square</Button>
  </div>
</div>
```
